# Phase 2: 시료 관리 (등록 / 조회 / 검색)

## 목표

시료를 시스템에 등록하고, 목록을 조회하거나 이름으로 검색할 수 있게 한다.

## 참고 문서

- [sample-management.md](../docs/features/sample-management.md)

## 구현 범위

1. **시료 등록**: 시료 ID, 이름, 평균 생산시간, 수율을 입력받아 새로운 시료를 추가
2. **시료 조회**: 등록된 모든 시료 목록과 현재 재고 수량을 함께 표시
3. **시료 검색**: 이름 등 속성으로 특정 시료를 검색

## 고려 사항

- 시료 등록 시 초기 재고 수량 처리 방식은 아직 미결 사항이다
  ([sample-management.md](../docs/features/sample-management.md#미결-사항) 참고). 구현 전에
  정책을 정해야 하며, 정한 내용은 문서에 함께 반영한다.
- Phase 1에서 만든 저장소 계층을 사용해 등록한 시료가 영속화되는지 함께 검증한다.

## 완료 기준 (체크리스트)

- [ ] 시료 등록 성공/중복 ID 등록 시도에 대한 테스트 작성 및 통과
- [ ] 시료 조회(전체 목록 + 재고 수량 표시) 테스트 작성 및 통과
- [ ] 이름 기반 검색(일치/불일치/부분 일치 등) 테스트 작성 및 통과
- [ ] 초기 재고 수량 정책 결정 및 관련 문서 Update
